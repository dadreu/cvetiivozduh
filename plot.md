# План разработки проекта (Roadmap)

## ЭТАП 1: Инициализация и настройка инфраструктуры бэкенда
- [ ] Инициализировать проект ASP.NET Core Minimal API (.NET 8/9).
- [ ] Настроить структуру проекта (папки Models, DTOs, Endpoints, Data).
- [ ] Настроить подключение к MySQL (`Pomelo.EntityFrameworkCore.MySql`) с использованием `AddDbContextPool` и `ServerVersion.AutoDetect`.
- [ ] Настроить конфигурацию Cookie-аутентификации.
- [ ] Настроить Data Protection (сохранение ключей на файловую систему).
- [ ] Добавить ForwardedHeadersMiddleware (для корректной работы за Nginx).

## ЭТАП 2: База данных и модели
- [ ] Создать сущности базы данных (Flower, Category, User/Admin).
- [ ] Создать DTO (Data Transfer Objects) для выдачи списка товаров и карточки товара.
- [ ] Создать первую EF Core миграцию и применить её (Code-First).
- [ ] Написать скрипт для первичного заполнения БД (Seeding) учетной записью администратора.

## ЭТАП 3: Реализация API
- [ ] **Auth API:** Эндпоинты `/api/auth/login` и `/api/auth/logout`. Реализация входа с выдачей куки.
- [ ] **Anti-forgery API:** Эндпоинт `/api/antiforgery/token` для выдачи CSRF токена фронтенду.
- [ ] **Public API (Каталог):** Эндпоинты для получения списка цветов и детальной информации. Использование `.AsNoTracking()`.
- [ ] **Admin API (Управление):** Защищенные маршруты (требуют авторизации) для добавления, редактирования и удаления товаров.
- [ ] **Upload API:** Реализация безопасной загрузки изображений товаров (проверка размера, генерация Guid, сохранение на диск вне папки приложения).

## ЭТАП 4: Инициализация и настройка фронтенда
- [ ] Создать Vue 3 проект через Vite.
- [ ] Установить и настроить Vue Router, Pinia.
- [ ] Установить и настроить Axios (интерцепторы, `withCredentials`).
- [ ] Настроить `vite-plugin-prerender` для SEO.

## ЭТАП 5: Дизайн-система и верстка (Скандинавский минимализм)
- [ ] Настроить глобальные CSS/SCSS переменные для цветовой палитры (базовые, вторичные, акцентные цвета).
- [ ] Настроить типографику и отступы.
- [ ] Создать базовые UI-компоненты (Кнопки, Инпуты, Карточки товара).

## ЭТАП 6: Клиентская часть (Лендинг)
- [ ] Разработать главную страницу (каталог товаров).
- [ ] Интегрировать получение данных с Public API.
- [ ] Настроить обработку изображений (пути к `/uploads/`).

## ЭТАП 7: Клиентская часть (Админ-панель)
- [ ] Разработать страницу авторизации администратора.
- [ ] Настроить Navigation Guards в Vue Router для защиты админских путей.
- [ ] Разработать дашборд управления товарами (список, добавление, редактирование).
- [ ] Интегрировать загрузку файлов с передачей Anti-forgery токена.

## ЭТАП 8: Инфраструктура и деплой

- [ ] Финальное тестирование связки.
